# 10 Asset System

## 定位

Creative OS 保存的是资产，不是一次性结果。

资产类型：

- character
- skeleton
- animation
- image
- video
- text
- document
- project

## 第一阶段资产

- Character Asset
- Skeleton Asset
- Animation Asset
- Preview Asset

## 资产原则

- Asset 可被多个 Node 引用
- Asset 可复用
- Asset 可导入 / 导出
- Asset 与 Project 可分离
