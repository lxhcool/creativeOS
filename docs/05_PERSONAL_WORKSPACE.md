# 05 Personal Workspace Principles

## 定位

Creative OS 第一阶段是个人创作者工作台。

默认：

```text
User
└── Personal Workspace
    ├── Projects
    └── Assets
```

## 不做

- 多租户
- 团队空间
- 组织管理
- RBAC
- 企业审计

## 必须做

- 注册 / 登录
- 个人工作区
- 项目列表
- 项目保存
- 自动保存
- 导入项目
- 导出项目
- 资产库

## 优先级

```text
Project Safety > Node System > Agent Workflow > UI Polish > Enterprise Features
```
