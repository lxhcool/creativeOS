# 01 Vision

## 产品定位

Creative OS = Personal AI Infinite Canvas Workspace。

它不是单一的 AI 游戏资产工具，也不是企业级多租户 SaaS。  
它是一个面向个人创作者的 AI 创作工作台。

长期支持：

- Game Asset Studio
- Image Studio
- Video Studio
- Writing Studio
- PPT / Document Studio
- Agent Workflow Studio

## 第一阶段定位

第一阶段选择 Game Asset Studio 的角色资产生成功能作为验证模块。

原因：角色资产生成可以同时验证：

- Infinite Canvas
- Node System
- Agent Workflow
- Asset System
- Project System
- Preview Runtime

## 第一阶段不是最终产品边界

Game Asset Studio 只是 Creative OS 的第一个模块，不是整个产品本体。
